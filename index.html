Integration of Specs2 with the Scala IDE

What's this project all about?

Linking Specs2 (video) into the ScalaTest integration of Scala IDE.

  • Select a Specs2 file/suite/test from source code to run

Limitations

  • Several libraries are required on project level to make the plugin work. We're working on it to reduce them.
  • You have to add a specific annotation, @WrapWith(classOf[Specs2Runner]) before all of the specifications you wold like to run with the ScalaTest plug-in. We are also working on the issue with Bill Venners, Chua Chee Seng and Eric Torreborre (it requires extraction of some abstractions from ScalaTest and a slight modification of the Specs2 API).
  • Auto-formatting of acceptance specifications is working with the default behaviour of scalariform. We are working on Specs2-specific extensions.
